# 🏏 Hand Cricket Game

A browser-based Hand Cricket game that uses real-time hand gesture recognition with TensorFlow.js and the Handpose model. Players use their fingers to play against the computer through their webcam.

## Features
- 🎥 Real-time hand gesture detection
- 🏏 Odd-Even toss to decide batting or bowling
- 🤖 Play against the computer
- 📊 Live score tracking
- 🎯 Target-based second innings
- 🔄 Play Again option

## Technologies Used
- HTML
- CSS
- JavaScript
- TensorFlow.js
- Handpose Model
- WebRTC (Camera Access)

## How to Run
1. Clone this repository.
2. Open `index.html` in your browser.
3. Allow camera access.
4. Start playing!
